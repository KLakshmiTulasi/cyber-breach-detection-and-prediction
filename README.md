# Cyber Breach Detection and Prediction using Machine Learning

## Overview

Cybersecurity threats such as phishing websites and malicious URLs continue to pose significant risks to individuals and organizations. Early detection of such threats can help prevent data breaches, financial losses, and unauthorized access.

This project uses Machine Learning techniques to classify URLs as legitimate or malicious and provides predictions through a Flask-based web application.

## Problem Statement

Develop a machine learning system capable of detecting potentially harmful URLs and predicting cyber threats using URL-based features.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* TF-IDF Vectorization
* Random Forest
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* Flask
* Jupyter Notebook

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Extraction using TF-IDF
4. Model Training
5. Model Evaluation
6. Model Comparison
7. Model Deployment using Flask

## Machine Learning Models

* Random Forest
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score

## Results

The models were evaluated and compared using classification metrics. The final model achieved high accuracy in distinguishing legitimate websites from phishing or malicious URLs.

## Deployment

A Flask web application was developed to allow users to enter URLs and receive predictions in real time.
